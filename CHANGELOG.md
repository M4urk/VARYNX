# Changelog

## [0.1.0] – 2026-02-14
### Added
- Initial closed beta release
- Full Android UI (15+ screens)
- GuardianService background process
- Python guardian engine (modular detectors)
- Call Protection module
- Message Scanner module
- Crypto Shield module
- Permission Guard (overlay detection)
- Bluetooth Proximity Scanner
- Guardian Score dashboard
- Threat History system
- Lockdown Mode UI
- Child Mode UI
- Branding, logo, and theme
- GitHub documentation and structure

### Notes
This release focuses on UI, architecture, and early user feedback.  
Real‑time engine → UI integration will arrive in the next phase.

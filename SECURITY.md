# Security Policy

## Supported Versions
VARYNX is currently in closed beta.  
Security patches will be applied to the latest version only.

## Reporting a Vulnerability
If you discover a security issue, please report it responsibly.

### Do NOT:
- Open a public GitHub issue
- Share the vulnerability publicly

### Instead:
Email us privately at:

**varynx.security@proton.me**

Include:
- Description of the issue
- Steps to reproduce
- Potential impact
- Suggested fix (optional)

We will acknowledge your report within 72 hours.

## Philosophy
VARYNX is a defensive‑only platform.  
We do not support offensive security research or exploit development.
